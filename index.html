Introduction

JavaScript was created by Netscapeâ€™s Brendan Eich in 1995. It was originally intended as a simpler scripting language for websites, complementing the use of Java for more complex web applications, but its tight integration with Web pages and built-in support in browsers has caused it to become far more common than Java in web frontends.

  • JavaScript is a lightweight, interpreted programming language.
  • It is designed for creating network-centric applications.
  • JavaScript is very easy to implement because it is integrated with HTML.
  • It is open and cross-platform.
Prerequisites

For get started with JavaScript usually you don't need to have any prior experience with other programming language. But to use JavaScript for the web you should familiar with the following:

  1. HTML
  2. CSS

If you are familiar with C or Java type of language then it will be a big plus for you.

Comments

You can commenting out lines in JavaScript either single or multi line.If you want to use single line comment then use ' // ' and if you are interested for multi line commenting then use ' /*...*/ '.

For Example:

getStarted(); // single line comment getStarted(a, b); /* a and b are the two parameters with data type numbers. and it initialize the parameters to some numbers. */
